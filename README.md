# SkillZone

Este projeto foi gerado e pode ser gerenciado de várias maneiras.

## Como posso editar este código?

Você pode clonar o repositório e trabalhar localmente com sua IDE preferida.

É necessário ter Node.js e npm instalados - [instale com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga estes passos:

```sh
# Passo 1: Clone o repositório.
git clone <URL_DO_SEU_GIT>

# Passo 2: Navegue até o diretório do projeto.
cd skillzone

# Passo 3: Instale as dependências necessárias.
npm i

# Passo 4: Inicie o servidor de desenvolvimento.
npm run dev
```

## Tecnologias utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
